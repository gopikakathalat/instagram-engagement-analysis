# Instagram Engagement Behaviour Analysis

## Project Overview
This project analyzes personal Instagram interaction data to understand engagement behaviour patterns. The dataset includes liked posts, story likes, followers, and following activity exported from Instagram.

## Objectives
- Identify the most frequent interaction types
- Analyze peak Instagram activity times
- Determine top accounts engaged with
- Study engagement trends over time

## Tools Used
- Google Sheets – Data cleaning
- MySQL – Data analysis
- Tableau Public – Data visualization

## Key Visualizations
- Interaction Distribution
- Top Accounts Engaged
- Activity by Day
- Instagram Activity Heatmap
- Engagement Trend
- Followers Trend
- Following Trend Analysis

## Key Insights
